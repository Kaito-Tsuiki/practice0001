①gitのリポジトリをクローンする---------------------------------------------------------------------------------------
以下のリポジトリをクローン
https://github.com/Kaito-Tsuiki/test0001.git

②vender以下のファイルをDLする----------------------------------------------------------------------------------------
以下のコードを順に実行する
cd src\NWS
composer update

③必要なソース情報を構築する------------------------------------------------------------------------------------------
git pull

④docker情報を構築する------------------------------------------------------------------------------------------------
dockerのインストール後以下のコード実行
docker-compose up -d --build

⑤DB情報を構築する----------------------------------------------------------------------------------------------------
以下のコードを順に実行する
docker exec -it php-fpm bash
cd NWS
php artisan migrate
