１.gitのリポジトリをクローンする---------------------------------------------------------------------------------------
以下のリポジトリをクローン
https://github.com/Kaito-Tsuiki/test0001.git

２.vender以下のファイルをDLする----------------------------------------------------------------------------------------
以下のコードを順に実行する
cd src\NWS
composer update

３.必要なソース情報を構築する------------------------------------------------------------------------------------------
git pull

４.docker情報を構築する-----------------------------------------------------------------------------------------------
dockerのインストール後以下のコード実行
docker-compose up -d --build

５.DB情報を構築する---------------------------------------------------------------------------------------------------
以下のコードを順に実行する
docker exec -it php-fpm bash
cd NWS
php artisan migrate
